Proyectos

Portal de Gestión de Talleres Mecánicos (TFG)

Plataforma full-stack para la gestión completa de talleres mecánicos y mecánicos freelance. Incluye sistema de citas, gestión de clientes y vehículos, control de roles (admin/empleado), panel de administración, y autenticación segura con JWT. El proyecto integra API REST desarrollada en Java con arquitectura en capas, conectada a PostgreSQL, y un frontend en Vue. Permite dar de alta talleres como administrador, visualizar citas, gestionar disponibilidad y administrar empleados del taller. Como usuario normal, puedes agregar tu vehículo y gestionar citas para los diferentes talleres que están dados de alta.

Java · REST API · JWT · PostgreSQL · Vue 3 · Ionic · Vite · Pinia · TypeScript · Node.js

Intranet empresarial

Intranet full-stack desarrollada para centralizar la gestión interna de una empresa. Incluye autenticación con JWT, control de roles (admin/empleado),cifrado de contraseñas, notificaciones internas, panel de administración, y CRUD completo de empleados con filtros y permisos avanzados. El sistema solo le permite al administrador crear nuevos empleados, gestionar y cambiar sus datos, subir las nóminas de cada empleado y gestionar las peticiones de vacaciones de cada empleado. Construido con arquitectura en capas, base de datos relacional y frontend moderno con diseño responsive orientado a móviles.

Java · REST API · JWT · PostgreSQL/MySQL · Vue 3 (HTML5, CSS, JavaScript) · Vite · Pinia · Ionic · TypeScript · Node.js

Repositorio (GitHub)

Plano Interactivo

Módulo visual que representa el plano de una instalación de forma interactiva usando Vue y Konva. Cada sala se dibuja como un área independiente con sus sensores de temperatura y humedad, que se actualizan en tiempo real mediante SSE. Además, permite controlar la iluminación de cada zona: al pulsar el botón central se abre un menú de confirmación y, si se activa la luz, la sala se resalta con un efecto LED y degradados que simulan iluminación real sobre el plano.

Vue 3 · vue-konva · EventSource (SSE) · JavaScript · CSS
